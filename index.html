Introducción

Introducción

¿Qué es Flutter?

Flutter es un kit de desarrollo de software (SDK) de código abierto creado por Google para el desarrollo de interfaces de usuario multiplataforma.

Introducción

¿Qué problema resuelve?

Flutter resuelve varios problemas comunes en el desarrollo de aplicaciones, como la necesidad de crear aplicaciones nativas para múltiples plataformas con un solo código base.

Arquitectura general

Arquitectura general

Flutter se organiza en tres capas principales:

  • El framework
  • El engine
  • El embedder

Arquitectura general

El framework

Es la parte donde los desarrolladores suelen trabajar directamente.

Arquitectura general

El engine

El engine esta escrito en C++ y se encarga del renderizado, la composición de la UI, el acceso al sistema de archivos y la ejecución del código compilado de Dart

Arquitectura general

El embedder

Es la capa específica de cada plataforma se encarga de integrar Flutter con el sistema operativo anfitrión:

  • gestiona la ventana de la aplicación
  • los eventos del sistema
  • la comunicación nativa

Widgets y estructura

Widgets y estructura

Los widgets son los bloques de construcción fundamentales para crear interfaces de usuario. Estos widgets se pueden combinar, anidar y personalizar .

Existen dos tipos principales de widgets en Flutter:

  • StatelessWidget
  • StatefulWidget

Widgets y estructura

StatelessWidget

Un widget cuyo estado no cambia durante el ciclo de vida de la aplicación. Ideal para vistas que no requieren actualizaciones dinámicas.

Widgets y estructura

StatefulWidget

Un widget cuyo estado puede cambiar durante el ciclo de vida. Utiliza un mecanismo de gestión del estado para actualizar la UI cuando los datos cambian.

Widgets y estructura

Estructura del árbol de widgets

Determina cómo Flutter renderiza la interfaz. Los widgets se organizan jerárquicamente, y la UI se vuelve más eficiente a medida que se optimiza el proceso de "reconstrucción" de las vistas.

Lenguaje Dart

Lenguaje Dart

Dart es un lenguaje de programación desarrollado por Google, y es el lenguaje principal detrás de Flutter. Diseñado para ser rápido, eficiente y fácil de aprender.

Lenguaje Dart

Dart soporta tanto compilación Just-In-Time (JIT) durante el desarrollo, lo que permite un ciclo de desarrollo rápido con Hot Reload, como también Ahead-Of-Time (AOT) para la compilación de código optimizado en producción.

Lenguaje Dart

Características claves

  • Orientación a objetos: Dart sigue un modelo orientado a objetos con clases y objetos.
  • Null Safety: Dart tiene un sistema de seguridad contra valores nulos que ayuda a prevenir errores comunes y mejorar la confiabilidad del código.
  • Futuro y asincronía: Dart tiene un sistema integrado para manejar operaciones asíncronas con async y await.

Rendimiento y herramientas

Rendimiento

Flutter destaca por su rendimiento gracias a su motor gráfico Skia, que permite renderizar interfaces rápidamente sobre el lienzo de la plataforma, superando las limitaciones de los componentes nativos del sistema operativo.

Flutter ofrece un conjunto de herramientas para facilitar el desarrollo y la depuración de aplicaciones:

                  class Counter extends StatefulWidget {
                    @override
                    _CounterState createState() => _CounterState();
                  }
                  
                  class _CounterState extends State<Counter> {
                    int _counter = 0;
    
                    void _incrementCounter() {
                      setState(() {
                        _counter++;
                      });
                    }

Cierre y futuro
